# 🐍 Snake Game using Python

A classic Snake Game built using Python and Tkinter. Control the snake using arrow keys, eat food to grow longer, increase your score, and avoid colliding with walls or yourself.

## 📌 Project Overview

This project is a graphical implementation of the popular Snake Game using Python's Tkinter library. The game features smooth movement, score tracking, food generation, collision detection, and a simple user interface.

The objective is to control the snake, collect food, grow longer, and achieve the highest score possible without hitting the boundaries or the snake's own body.
